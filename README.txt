TAREA 2 ALGORTIMOS Y COMPLEJIDAD

* Compilar:
- Abrir terminar de linux en la carpeta donde esté el makefile y los archivos con extensión .cpp

- Para compilar los códigos se debe ejecutar el siguiente comando en terminal:

                                    $ make

- Para borrar los ejecutables y archivos con extensión .o, se debe ejcutar en la terminal:

                                    $ make clear

* Ejecución:
- Para ejecutar el código coorespondiente a la parte 1 con el algoritmo de Dividir y Conquistar, se debe ejecutar en la terminal:

				    $ make ejecutarDC 

- Para ejecutar el código coorespondiente a la parte 1 con el algoritmo de Fuerza Bruta, se debe ejecutar en la terminal:

				    $ make ejecutarFB

- Para ejecutar el código coorespondiente a la parte 2, se debe ejecutar en la terminal:

				    $ make ejecutar2

* Consideración:
- El programa se probó utilizando el compilador g++ versión 7.4.0 en una terminal de ubuntu versión 18.04.1
